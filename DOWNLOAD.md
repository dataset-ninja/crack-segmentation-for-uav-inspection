Dataset **Crack Segmentation for UAV Inspection** can be downloaded in Supervisely format:

 [Download](https://assets.supervise.ly/supervisely-supervisely-assets-public/teams_storage/Y/S/SY/e0rC2557N5DFd2xvV44lMxf1YhN8XwQIKcibtmMJZM7XsfsjKbqyjAhCY8jJHLEew7dtTcGTmauSSRn2o1c1KhX7bUZhd8cxnbWqoDteMNJWzYswdgY736NG0IdW.tar)

As an alternative, it can be downloaded with *dataset-tools* package:
``` bash
pip install --upgrade dataset-tools
```

... using following python code:
``` python
import dataset_tools as dtools

dtools.download(dataset='Crack Segmentation for UAV Inspection', dst_path='~/dtools/datasets/Crack Segmentation for UAV Inspection.tar')
```
